# cucumber_automation

How to run test suite:

1) clone project from github
2) setup Java 8 JDK and latest maven 
3) run test suite with the command 'mvn clean install' in command line
4) test report will be stored in target/cucumber-reports/cucumber-html-reports/.
To review the result open overview-features.html report file

Note: tests can be run separately with junit mechanism with RMB against the feature/Scenario -> Run
